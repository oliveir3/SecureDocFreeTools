# SecureDocFreeTools

SecureDocFreeTools is a **free, browser-based suite** of tools for PDFs, images, text, and files. All tools run **entirely in your browser** ‚Äî no server, no API keys, no paid plans. Fully **offline-capable** after page load, ensuring privacy and security.  
Aligned with SecureDoc‚Äôs principles: fast, reliable, and secure digital workflows for everyone.

---

<details>
<summary>üåü Features</summary>

### Text & OCR Tools
- **Image to Text (OCR)** via Tesseract.js  
- **PDF to Text** using PDF.js + PDF-LIB  
- **Text to Speech / Speech to Text** (Web Speech API)  
- **Text Cleaning** (remove line breaks, extra spaces, special characters)  

### PDF Tools
- Merge PDFs, Split PDFs, Compress PDFs (PDF-LIB + pako.js)  
- Extract Images, Add Text/Images to PDFs  

### Image Tools
- Crop & Resize (Cropper.js)  
- Compress Images (browser-image-compression)  
- Convert Formats (JPG, PNG, WebP)  
- Apply Filters (grayscale, blur, brightness)  

### File Conversion Tools
- DOCX ‚Üí PDF, HTML ‚Üí PDF, Text/Markdown ‚Üí PDF  
- ZIP files: compress & extract  

### Barcode & QR Code Tools
- Generate QR Codes (qrcode.js)  
- Scan QR/Barcodes (jsQR + QuaggaJS)  

### Utility Tools
- Word & Character Counter  
- JSON Formatter/Beautifier  
- Base64 Encode/Decode  
- CSV ‚Üî JSON  
- HTML Cleaner & Minifier  

### Document Extraction & Conversion
- Extract Tables from PDF ‚Üí CSV (PDF.js + PapaParse)  
- Extract Text from DOCX ‚Üí HTML (mammoth.js)
